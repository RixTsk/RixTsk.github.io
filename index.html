Tilda - не просто конструктор сайтов, это полноценная платформа для создания и размещения веб-сайтов, которая включается в себя как редактор (с уже включенными и настроенными стандартными блоками), так и возможность создавать индивидуальные блоки для сайта (Zero - блоки).

Из существенных плюсов можно отметить:

  • скорость разработки сайта (особенно на стандартных блоках);
  • возможно размещение сайта непосредственно на платформе;
  • в стоимость пользования сервисом включен SSL сертификат;
  • относительная простота в дальнейшей поддержке сайта...
Однако, у данной платформы есть и некоторые минусы:
  • разработанные сайты весьма посредственно оптимизированны (в плане CSS / JS кода);
  • Zero-блоки иногда “пытаются” съехать с указанных позиций;
  • на бюджетных тарифах нет возможности “вытащить” сайт с платформы;
  • годовая стоимость подписки...
Имея определенный опыт в разработке сайтов, большую часть минусов можно с легкостью нивелировать.
Плафторма хорошо подходит для разработки и поддержания следующих типов сайтов:
  • сайт-визитка;
  • сайт-блог.

TILDA

Webflow - еще один конструктор сайтов, но в отличии от Tilda, Webflow требует понимания принципов работы с HTML, CSS, а также логики языков программирования (JavaScript и PHP). Однако, такие особенности дают большую гибкость для работы с данным инструментом, при этом результатом работы становится сайт со сравнительно "чистым кодом".

Из существенных плюсов можно отметить:

  • относительно высокая скорость разработки сайтов;
  • сравнительно "чистый код", хорошая оптимизация разработанного сайта;
  • имеются базовые инструменты для SEO оптимизации будущего сайта;
  • упрощение работы по адаптации сайтов под мобильные устройства...
А теперь о некоторых минусах платформы:
  • при публикации сайта, необходимо в ручную "подчищать" HTML файл;
  • нет встроенного редакторка для создания программной части сайта;
  • "вытащить сайт" с платформы, можно только на платных тарифах;
  • стоимость подписки...
Не смотря на все минусы данной платформы, Webflow - прекрасный инструмент, с помощью которого можно создать хорошо оптимизированный сайт.
Плафторма хорошо подходит для разработки и поддержания следующих типов сайтов:
  • сайт-визитка;
  • сайт-блог;
  • лендинг;
  • многостраничный сайт.

WEBFLOW

Custom Deisgn - разработка сайта в "ручном" режиме с использованием различных инструментов. Например: язык разметки - HTML, язык стилей - CSS, языки программирования - JavaScript и PHP. Основное приемущество, перед разработкой сайтов в редакторах - абсолютная свобода и гибкость в процессе создания сайтов, что в свою очередь позволяет реализовывать самые разнообразные пожелания в плане функциональности и дизайна.

Из существенных плюсов можно отметить:

  • гибкости при работе с инструментами;
  • нет привязки к какой-либо платформе, сайт легко можно перемещать между хостингами;
  • нет подписок;
  • при определенных навыках - простота в дальнейшей поддержке сайта...
А теперь и некоторые минусы:
  • необходимо самостоятельно искать хостинг и приобретать доменное имя для сайта;
  • SSL сертификаты приобретаются отдельно;
  • поддержка сайта требует определенных навыков и квалификации;
  • разработка и настройка сайта занимает большее количество времени...

Учитывая возможности и доступную вариативность в данном варианте создания сайтов, ограничений по типам / дизайну / функционалу - практически нет.

CUSTOM DESIGN

После того как создание сайта подошло к логическому концу, приобретено доменное имя и определен хостинг, приходит пора задуматься о поддержке и продвижении сайта. С учетом количества сайтов, которые ежедневно регистрируются в сети, индексация в поисковых системах занимает определенное время (до 14 календарных дней), поэтому мгновенного потока пользователей в день публикации сайта ожидать не стоит. Однако, если не предпринимать никаких мер по продвижению сайта, пользователей и вовсе можно не дождаться.

Для привлечения пользователей на сайт, как правило, используются 2 технологии:

  • SEO продвижение
  • Контекстная реклама
Обе технологии эффективны, но в различных временных периодах. К примеру, SEO продвижение имеет накопительный эффект, который достигается за счет оптимизации сайта, расстановки необходимых тегов в файле HTML, наращивания "ссылочной массы" сайта, а также - создания семантического ядра (которое зависит от тематики сайта, а также желаемых результатов). В отличии от Контекстной рекламы, стоимость SEO продвижения более чем демократична.

Контекстная реклама показывает незамедлительный эффект, однако требует постоянного мониторинга и реагирования на изменение ситуации с переходами по рекламным ссылкам. Стоимость контекстной рекламы может быть от пары тысяч в месяц, до десятка тысяч за день, в зависимости от желаемого результата по привлечению новых пользователей на сайт.
